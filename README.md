# 🚀 Media Vault

Media Vault is a lightweight web application that allows you to save, organize, and manage videos, reels, and links from multiple platforms — all in one place.

⚡ Built using pure HTML, CSS, and JavaScript with localStorage (no backend required).

---

## 🎯 Features

- 🔗 Save links from multiple platforms
- 🎥 YouTube thumbnail + video title auto-fetch
- 🧠 Smart platform detection (YouTube, Instagram, Direct Video, Others)
- 🗂️ Custom categories (e.g., Education, Coding, Entertainment)
- 🔍 Real-time search and filtering
- ❤️ Favorite (like) system
- 🧩 Drag & drop card reordering
- 📦 Import & Export saved data (JSON)
- 🌙 Dark / Light mode toggle
- ⚡ Fully responsive design

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- Vanilla JavaScript  
- LocalStorage (for data persistence)
